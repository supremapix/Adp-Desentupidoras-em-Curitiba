# Imagens ADP Desentupidora

Este diretório contém as imagens reais da ADP Desentupidora utilizadas no site.

## Imagens Necessárias

### tecnico-adp-servico.png
- **Uso**: Página inicial, seção "Por que Curitiba confia na ADP?"
- **Dimensões recomendadas**: 600x600px ou superior
- **Descrição**: Foto de um técnico da ADP em ação, uniformizado, realizando serviço de desentupimento
- **Formato**: PNG ou WEBP para melhor qualidade

### atendimento-local.png
- **Uso**: Páginas de localização (cidades e bairros)
- **Dimensões recomendadas**: 800x400px ou superior
- **Descrição**: Foto de atendimento local, van da empresa, ou equipe técnica
- **Formato**: PNG ou WEBP para melhor qualidade

## Instruções

1. Substitua os arquivos placeholder por imagens reais de alta qualidade
2. Mantenha os mesmos nomes de arquivo
3. Otimize as imagens antes de fazer commit (use ferramentas como TinyPNG ou Squoosh)
4. Prefira formatos modernos como WEBP para melhor performance
